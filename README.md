# 📒 算术练习本（MathChild）

一个面向**小学中高年级**的算术练习卷生成器，纯前端静态网页，打开即用，支持 **A4 打印 / 导出 PDF**，**手机/平板/桌面全适配**，可一键部署到 GitHub Pages 分享。

## ✨ 功能特性

- **四种运算**：加、减、乘、除，可任意组合
- **三档难度**：入门（2 位数）/ 标准（3 位数）/ 进阶（4 位数）
- **两种题型**：
  - 横式（口算）—— 按难度自适应列数（2-3 位 5 列 / 4 位 4 列），可选 12/16/20 行
  - 竖式（列式）—— 4 列布局，固定 28 题每页，留足列式书写空间
- **智能约束**：
  - 减法结果不为负（被减数 ≥ 减数）
  - 除法允许有余数（自动保证余数 < 除数）
- **答案切换**：一键显示/隐藏答案，方便家长批改
- **响应式**：桌面侧栏 / 平板抽屉 / 手机 A4 纸张自动缩放适配屏幕
- **A4 打印**：自动分页，隐藏控件，含姓名/日期/得分栏
- **零依赖**：单 HTML + CSS + JS，无框架、无后端、数据不出本机

## 🚀 本地使用

无需安装任何环境，直接双击打开 `index.html` 即可在浏览器中使用。

> 推荐使用 Chrome / Edge 以获得最佳打印效果。

## 🖨️ 打印 / 导出 PDF

1. 设置好参数，点击 **「🎲 生成新卷」**
2. 点击 **「🖨️ 打印 / 导出 PDF」**（或按 `Ctrl + P`）
3. 在打印对话框中：
   - 目标打印机选「另存为 PDF」即可得到 PDF 文件
   - 纸张选 **A4**，边距选「默认」即可
   - 建议关闭「页眉页脚」
4. 点打印 → 完成 ✅

## 📦 部署到 GitHub Pages

1. **初始化并推送代码**

   ```bash
   git init
   git add .
   git commit -m "init: 算术练习生成器"
   git branch -M main
   git remote add origin https://github.com/<你的用户名>/MathChild.git
   git push -u origin main
   ```

2. **开启 GitHub Pages**

   - 进入仓库 → **Settings** → **Pages**
   - **Source** 选择 `Deploy from a branch`
   - **Branch** 选 `main`，文件夹选 `/ (root)`
   - 点 **Save**

3. **访问站点**

   等待 1~2 分钟，打开：

   ```
   https://<你的用户名>.github.io/MathChild/
   ```

## 🎯 难度档位说明

| 档位 | 加减法 | 乘法 | 除法 |
|------|--------|------|------|
| ★ 入门 | 2 位数 | 1~2 位数 | 1~2 位数 |
| ★★ 标准 | 3 位数 | 2 位数 | 2 位数 |
| ★★★ 进阶 | 3~4 位数 | 2~3 位数 | 2~3 位数 |

## 💡 使用建议

- **低年级**：选入门档 + 仅加减，横式 50 题，开启「减法结果不为负」
- **中年级**：标准档 + 全部运算，横式 50 题，开启「除法允许有余数」
- **高年级**：进阶档 + 全部运算，竖式 16 题，练习列式计算
- 想做错题本？用「显示答案」后打印一份带答案的对照版

## 📝 技术说明

- 除法采用**反向构造法**：先取商 q 和除数 d，再算被除数 = q×d + 余数，确保每道除法题都合法且余数恒小于除数
- 打印样式通过 `@media print` 和 `@page { size: A4 }` 控制，保证排版整洁、不跨页断题
- **所见即所得**：屏幕上就以真实 A4 尺寸（210mm × 297mm）显示纸张，深灰背景衬托白纸，打印效果与预览完全一致；横式 80 题、竖式 28 题均经过实测，正好填满一张 A4 单页
- 所有计算与渲染均在浏览器本地完成，无网络请求，无数据上传
